무제한 스토리지 및 동기화

크기 제한 없이 모든 파일 형식을 저장하세요. 모든 디바이스에서 자동 동기화되어 항상 최신 버전을 바로 이용할 수 있습니다.

  • 무제한 용량 파일당 또는 전체 스토리지 제한 없이 조직에 필요한 모든 것을 저장
  • 스마트 동기화 로컬 저장소를 채우지 않고 온디맨드로 파일 접근. 열 때만 다운로드
  • 데스크톱 및 모바일 앱 Windows, macOS, Linux, iOS, Android용 네이티브 앱과 오프라인 접근 지원
  • 대용량 파일 지원 최대 100GB 파일 업로드, 재개 가능한 업로드 및 무결성 검증 제공
  • 자동 백업 컴퓨터의 지정한 폴더를 클라우드로 지속적 백업

  • 세분화된 권한 소유자, 편집자, 댓글 작성자, 뷰어 —谁能 무엇을 할 수 있는지 정확하게 제어

맥락, 콘텐츠, 업무 패턴을 이해하는 AI로 몇 초 만에 원하는 파일을 찾으세요. 스마트 제안이 적절한 파일을 바로 앞에 가져다줍니다.

  • 자연어 검색 말하듯이 검색 — "지난달 사라와의 미팅에서 나온 예산 스프레드시트"
  • 콘텐츠 검색(OCR) 전문 검색 및 OCR 인덱싱으로 문서, 스캔한 PDF, 이미지 내부를 검색
  • 스마트 제안 캘린더, 이메일, 업무 맥락을 기반으로 필요할 수 있는 파일을 AI가 추천
  • 자동 태깅 AI가 자동으로 파일을 분류하고 태그하여 쉽게 정리하고 발견

관리 및 배포

스토리지 인프라에 대한 완전한 제어. 자체 서버에 배포하고, 할당량을 관리하고, 정책을 적용하고, 기존 시스템과 연동하세요.

  • 프라이빗 배포 온프레미스, 프라이빗 클라우드 또는 하이브리드 — 모든 데이터를 자체 인프라 내에 보관
  • 스토리지 할당량 사용자별, 부서별, 프로젝트별 스토리지 한도 설정, 알림 및 자동 아카이빙 제공
  • 보관 정책 규정 준수를 위한 데이터 보관 규칙 구성 — 기간 또는 유형에 따라 자동 아카이브 또는 삭제
  • 마이그레이션 도구 Google Drive, OneDrive, Dropbox 또는 모든 WebDAV/S3 소스에서 원활하게 이전
  • API 및 WebDAV 완전한 REST API, WebDAV 프로토콜 지원, S3 호환 인터페이스 제공
